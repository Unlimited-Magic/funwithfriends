# funwithfriends.party

The warmer front door to LET'S PLAY (letsplay.party). A single static landing page aimed at
"games to play with friends online free no download" search/ad angles.

- Not a fork. All CTAs deep-link into the existing host/join flow at letsplay.party — the same
  rooms and the same API. This repo is a marketing surface only.
- Assets are referenced absolutely from letsplay.party, so this repo stays a single file.
- Canonical source also lives in Unlimited-Magic/rumpus at fwf/index.html (staged at
  https://letsplay.party/fwf/). Keep the two copies in sync.

Served via GitHub Pages, custom domain funwithfriends.party (see CNAME). GitHub Pages allows one
custom domain per repo, which is why this is separate from rumpus (rumpus is canonically letsplay.party).
